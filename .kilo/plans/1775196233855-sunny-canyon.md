| Priority         | Count | Focus                                   |

## P0: Critical Bugs (Fix Immediately)

### P0-1: Fix undefined `var(--line)` CSS variable

**File:** `src/styles.css`  
**Issue:** CalendarMonthGrid.tsx uses `var(--line)` which is never defined  
**Fix:** Add `--line: var(--pebble)` to `:root` (or use `var(--pebble)` directly in the component)

## P3: Refactoring (Larger Changes)

### P3-1: Split LoginScreen.tsx

Extract from 332 lines into:

- `src/features/auth/components/GoogleIcon.tsx`
- `src/features/auth/components/EmailAuthForm.tsx`

### P3-2: Decouple useAuth.ts

Extract profile/GCal logic from useAuth into `@/lib/authService.ts`:

- Move `getOrCreateSnapuppyCalendar` call out of auth flow
- Move profile fetching to separate service
- Move guest marking to separate service

### P3-3: Add sitter_id consistency in bookingService

**Issue:** `getBooking` and `deleteBooking` fetch don't filter by sitter_id  
**Action:** Add `.eq('sitter_id', sitterId)` to these queries for consistency

### P3-4: Fix EmptyState hardcoded color

**Location:** `src/components/ui/EmptyState.tsx`  
**Issue:** Uses `#8fb886` instead of `var(--sage)`  
**Fix:** Use CSS variable
